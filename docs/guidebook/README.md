# ditdah CW Learner Guidebook

A sound-first path from zero to useful CW.

This guide is for a complete beginner who wants to become a confident ditdah
user and then keep growing toward real on-air Morse code. It is deliberately
practical: what to do today, what to listen for, which ditdah feature to use,
and when to reach for outside practice.

## The Promise

By the end of this path you should be able to:

- Use every ditdah practice mode with intention.
- Copy individual characters, short groups, numbers, common words, callsigns,
  and simple QSO fragments by ear.
- Tune speed, spacing, tone, round length, strict mode, and references without
  accidentally training a bad habit.
- Use history, weak spots, daily goals, streaks, and optional cloud sync to steer
  your own practice.
- Continue beyond ditdah with live classes, code-practice files, contest
  simulators, and real QSOs.

This is not a license exam guide and it is not a visual code chart. ditdah's
core belief is simple: CW is a sound language. Learn the sound, respond to the
sound, and only use visual references when they support that habit.

## Sound-First Rules

1. Do not memorize from a visual pattern chart.
2. Keep character speed high enough that each character feels like one sound.
3. Slow the gaps before you slow the character itself.
4. Practice in short, repeatable sessions.
5. Misses are data, not failure.
6. Move from characters to chunks as soon as you can.
7. Add outside material only when it supports your current bottleneck.

The Koch method, used by ditdah's Learn mode, introduces a small set of
characters at real character speed and adds more as accuracy rises. Farnsworth
timing keeps the characters crisp while stretching the spacing so beginners
have time to respond.

## Day 0: Set Up ditdah

Open the live app: <https://dydx404.github.io/ditdah/>

Before your first round:

- Use headphones or a quiet speaker.
- Start with the default timing: 20 WPM character speed and 10 WPM effective
  speed.
- Leave strict mode on.
- Leave the visual pattern reference off unless you are deliberately checking a
  reference away from active copying.
- Keep answer sounds on if they help the loop feel immediate.
- Set the tone to a comfortable pitch. The default is fine for most learners.
- If you will practice on multiple devices, sign in only after you understand
  that ditdah is local-first and cloud sync is optional.
- Install the PWA if you want the app on your phone or laptop launcher. It also
  works offline after loading once.

Your first target is not speed. Your first target is a clean loop: hear, answer,
receive feedback, continue.

## The 8-Week Path

The weeks below are a guide, not a lock. Move faster when your history says you
are ready. Slow down when you notice guessing, counting, or frustration.

### Week 1: First Sounds

Use: Learn mode.

Daily plan:

- Do 3 rounds per day to meet the daily goal.
- Split practice into 2 or 3 short sessions if possible.
- Keep rounds short enough that you finish alert.
- Repeat missed characters immediately when strict mode asks for it.

What good practice feels like:

- You hear a character and answer before translating it in your head.
- You can miss one prompt without losing the next five.
- Your accuracy rises across the week without you needing a chart.

When to adjust:

- If you are overwhelmed, lower effective WPM first.
- If you are counting elements, raise character WPM back toward 20 and increase
  spacing instead.
- If the tone is tiring, change pitch or volume before changing the lesson.

Move on when:

- You can complete several rounds near 90 percent accuracy.
- New unlocks feel challenging but not random.

### Week 2: Short Memory

Use: Learn mode, Copy groups, Free training.

Daily plan:

- Start with one Learn round.
- Add one Copy groups round using the unlocked set.
- Finish with Free training on weak spots.

Why this matters:

Single-character copying builds recognition. Groups build short auditory memory.
Real CW does not wait for you to recover after every character, so you need a
little buffer.

How to use Free training:

- Pick Weak spots when a few characters keep causing misses.
- Pick Unlocked when you want a fair review.
- Pick Numbers or Punctuation only when you are intentionally drilling those
  sets.
- Avoid making a giant custom set too early.

Move on when:

- Groups are uncomfortable but playable.
- You can keep copying after a miss instead of freezing.

### Week 3: Numbers and Clean Accuracy

Use: Learn mode, Numbers, Copy groups, History.

Daily plan:

- Do one normal Learn or Copy groups round.
- Do one Numbers round.
- Review History after practice, not during practice.

Numbers are common in signal reports, times, frequencies, logs, and callsigns.
They deserve separate practice, but they should not replace letters.

Use History to ask:

- Which mode produced the most misses?
- Did accuracy fall late in the round?
- Are there one or two characters that keep appearing in misses?
- Is speed really the problem, or is the session too long?

Move on when:

- You can copy short number runs without changing your listening strategy.
- You can name your current weak spots without guessing.

### Week 4: Words and Head Copy

Use: Words, Learn, Free training.

Daily plan:

- Start with one Words round.
- Follow with a short Free training drill on weak spots.
- Keep one Learn round if you are still unlocking characters.

Words mode changes the game. You are no longer only copying isolated sounds; you
are building recognition of useful chunks. Do not worry if your accuracy dips at
first. That dip means the task is different.

Head-copy drill:

- Play a word round normally.
- In a later round, try to hold the whole word in memory before typing.
- If that becomes stressful, return to normal copying for the rest of the
  session.

Move on when:

- Some short words arrive as a whole sound shape.
- You can recover when a word is missed.

### Week 5: Callsigns

Use: Callsigns, Numbers, Free training.

Daily plan:

- Copy callsigns at your normal effective WPM.
- Drill numbers if digits are the weak part.
- Drill specific letters only if History says they are the weak part.

Callsigns are excellent CW practice because they are structured but not
predictable. They force letter-number-letter transitions and reward clean short
memory.

How to practice:

- Copy the whole callsign before judging yourself.
- If you miss the prefix, keep copying the rest.
- If numbers cause panic, run a brief Numbers round and return.

Move on when:

- You can copy several callsigns in a row without pausing after the digit.
- You can tolerate unknown combinations without trying to guess from context.

### Week 6: QSO Fragments

Use: QSO, Story Mode.

Daily plan:

- Start with one QSO round.
- Play one Story chapter or retry a chapter that felt rough.
- Use Free training only for the specific weak spots exposed by the chapter.

QSO practice introduces the shape of real contacts: calling, exchanging reports,
names, locations, acknowledgements, and closings. The goal is not to memorize a
script. The goal is to recognize common pieces quickly enough to stay present.

Story Mode adds context. Treat each chapter like a small radio scene:

- Listen first.
- Copy what the other station sends.
- Send your response when the story asks for it.
- Replay chapters to improve accuracy, not to rush completion.

Move on when:

- You understand the role of common exchange fragments.
- You can finish a story chapter without needing perfect accuracy.

### Week 7: Longer Listening

Use: Words, Callsigns, QSO, outside audio.

Daily plan:

- Keep ditdah as the warm-up.
- Add 5 to 10 minutes of outside audio at a comfortable level.
- Do not abandon ditdah's feedback loop; use it to repair weak spots.

Good outside sources:

- ARRL W1AW code practice files.
- Morse Code Ninja practice sets.
- LCWO drills.
- Live class material if you are enrolled in CW Academy or the Long Island CW
  Club.

How to listen:

- Pick audio slightly below your panic point.
- Copy what you can and let the rest go.
- Bring repeated misses back into ditdah Free training.

Move on when:

- You can practice from both ditdah and outside audio without changing your core
  listening habit.

### Week 8 and Beyond: Real CW

Use: ditdah for focused drills, outside tools for operating realism.

Next goals:

- Join a live class or practice group.
- Copy real callsigns at higher speed.
- Try simple QSO copy from recordings.
- Use contest simulators only when basic callsign copy feels stable.
- Prepare for a short, friendly on-air QSO.

ditdah remains useful after the beginner phase because it gives fast, controlled
reps. Use it for weak spots, warm-ups, numbers, callsigns, and story practice.
Use the wider CW ecosystem for social pressure, real-world noise, and operating
habits.

## Mode Playbook

### Learn

Best for: new characters and core recognition.

Use Learn when you want the app to manage the Koch progression. It starts small,
tracks accuracy, and unlocks new characters when you are ready.

Avoid:

- Repeating only easy rounds forever.
- Dropping character speed so low that the sound becomes separate pieces.
- Checking a visual chart after every miss.

### Copy Groups

Best for: memory, flow, and copying through mistakes.

Copy groups asks you to hold several characters before submitting. It is the
bridge between single-character recognition and real copy.

Try:

- Shorter groups when starting.
- Longer groups when accuracy is stable.
- One group round after one Learn round.

### Free Training

Best for: deliberate practice.

Use Free training when you know what you want to drill. The character picker lets
you choose supported characters, presets, unlocked characters, or weak spots.

Good uses:

- Drill the two or three characters you confuse.
- Review unlocked characters before a new Learn session.
- Paste custom text when you want to rehearse a specific vocabulary set.

Bad uses:

- Selecting everything on day one.
- Using it as a chart substitute.
- Practicing only your favorite characters.

### Numbers

Best for: digits in reports, times, logs, and callsigns.

Numbers mode isolates digits so you can stop treating them as a surprise inside
other practice.

### Words

Best for: chunking and head copy.

Words mode helps you hear common words as units. It is where copying starts to
feel less like typing individual letters.

### Callsigns

Best for: ham-radio realism.

Callsigns combine letters and numbers in compact, meaningful strings. They are
one of the best ways to prepare for real operating.

### QSO

Best for: exchange vocabulary.

QSO mode presents useful contact fragments. Use it when you are ready to hear
CW as communication rather than only as drills.

### Story Mode

Best for: motivation, context, and scenario practice.

Story chapters turn QSO fragments into small missions. Current chapters include
first contact, weather-net, and harbor-relay scenarios. Expect copy prompts,
send prompts, chapter completion, and replay.

### History

Best for: honest self-coaching.

History is for after the round. It shows recent practice so you can see trends
instead of relying on mood.

### Daily Goal and Streak

Best for: consistency.

The daily goal is intentionally small. Three rounds on a busy day beats one
heroic session followed by four days off.

## Settings Playbook

### Character WPM

Character WPM controls how fast each character is sent. Keep this high enough
that the sound arrives as one recognizable shape. For many beginners, 20 WPM is
a strong default.

### Effective WPM

Effective WPM controls the overall pace by stretching or shrinking the gaps.
Lower this when you need thinking room. Raise it when the silence starts to feel
too generous.

### Tone and Volume

Pick a tone you can listen to daily. A comfortable tone matters more than a
theoretically perfect tone.

### Round Length

Short rounds are not beginner mode; they are good training design. Increase
round length only when you can stay relaxed and accurate to the end.

### Strict Mode

Strict mode asks you to echo missed prompts before moving on. Leave it on while
learning. Turn it off only when you are deliberately practicing flow through
misses.

### Answer Sounds

Answer sounds make feedback immediate. If they annoy you, turn them off. If they
help practice feel like a game, keep them on.

### Visual Reference

The reference is opt-in and should be used like a dictionary, not like a lesson.
Do not copy while staring at it. If it becomes your crutch, close it.

### Language

ditdah currently supports English and Simplified Chinese. Switch the interface
language in settings; the learning method is the same.

### Account and Cloud Sync

ditdah is local-first. You can practice without an account. Sign in when you
want backup or cross-device sync, then let the app merge your local progress.

### Offline Install

Install ditdah as a PWA if you want fast access or offline practice. Load the
app once while online, then it can continue to work without the network.

## Troubleshooting

### I Keep Counting

Raise character speed or return to the default character speed. Lower effective
speed if needed. Counting usually means the sound is too slow or the visual
chart habit is taking over.

### I Freeze After a Miss

Use Copy groups and Story Mode to practice continuing. A missed character is
only one event. The next sound still deserves your attention.

### My Accuracy Drops Late in the Round

Shorten the round. Fatigue can look like a skill problem. Build endurance after
the recognition loop is stable.

### I Am Bored

Change mode, not philosophy. Try Words, Callsigns, QSO, or Story Mode. Keep the
practice sound-first.

### I Am Stuck on the Same Characters

Use Free training with Weak spots for a few minutes, then return to normal
practice. Do not spend the whole day on one confusion pair.

### I Can Copy in ditdah but Not Outside Audio

That is normal. Outside audio removes structure and feedback. Add outside audio
slowly, then bring the misses back into ditdah for focused repair.

### I Want to Send

Receiving comes first for most beginners because it builds the sound vocabulary.
Story Mode includes simple send prompts, and future ditdah work can expand
sending practice. When using a real key, focus on rhythm, spacing, and clean
copyable code rather than speed.

## Outside Resource Map

Use outside resources with a purpose. More material is not automatically better.

## Research Basis

The guidebook's practice path is based on a few recurring themes across CW
learning resources:

- The ITU recommendation is the formal International Morse reference, so ditdah
  treats standards as reference material rather than as a beginner lesson.
- ARRL points learners toward Koch/Farnsworth tools, W1AW practice files, and
  live code-practice schedules; W1AW audio is useful once the learner can handle
  longer listening without immediate feedback.
- CWops CW Academy and the Long Island CW Club both emphasize structured
  practice, live accountability, and regular assignments rather than occasional
  marathon sessions.
- LCWO, Morse Code Ninja, and Morse Code World all provide browser-accessible
  recognition drills, which makes them good companions to ditdah when a learner
  wants a second drill environment.
- RufzXP and Morse Runner are intentionally later-stage tools: they add speed,
  callsign pressure, and contest realism after the basic listening loop is
  stable.
- FOC, The Art and Skill of Radio-Telegraphy, and Zen and the Art of
  Radiotelegraphy are best treated as deep reading for operating habits,
  mindset, and long-term proficiency.

## How to Combine ditdah With Outside Practice

1. Use ditdah to build recognition and expose weak spots.
2. Use one outside resource to add realism or social practice.
3. Do not add three new tools in the same week.
4. Bring repeated misses back into Free training.
5. Keep a small daily habit even when you add longer sessions.

Example weekly mix after Week 6:

- Monday: Learn or Words plus weak spots.
- Tuesday: Callsigns plus numbers.
- Wednesday: QSO or Story.
- Thursday: W1AW or Morse Code Ninja audio.
- Friday: ditdah weak spots.
- Saturday: live class, club practice, or relaxed listening.
- Sunday: rest or one easy round to keep the habit alive.

## Proficiency Checklist

You are becoming a proficient ditdah user when you can:

- Choose the right mode for the day's goal.
- Explain the difference between character WPM and effective WPM.
- Raise or lower difficulty without destroying the sound-first habit.
- Use History and Weak spots to pick tomorrow's drill.
- Copy numbers and callsigns without panic.
- Copy common words as chunks.
- Follow a simple QSO or Story chapter.
- Practice offline or across devices without losing progress.
- Know which outside resource to use next and why.

You are becoming a proficient CW learner when you can:

- Stay relaxed after a miss.
- Keep listening instead of replaying the previous sound in your head.
- Recognize some common words and exchange fragments as units.
- Copy for understanding, not only for perfect typing.
- Practice consistently without needing every session to feel great.

## Notes for Contributors

Future guidebook additions should follow the same product law as the app:

- No character pattern charts.
- No lessons that teach visual translation as the main path.
- No screenshots that reveal hidden answers during an active copy task.
- Prefer sound-first exercises, scenario context, and honest self-coaching.

When adding new app features, update this guidebook if the feature changes how a
learner should practice.
